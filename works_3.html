ToDoアプリ

制作物概要 必須

Spring Boot 3.2.2 + Thymeleaf + MyBatis + H2 を使った、
ログイン必須の Todo 管理アプリです。
Web 画面と REST API の両方を提供し、添付ファイル、カレンダー表示、CSV エクスポート、管理者機能、監査履歴などが入っています。

使用技術 必須

フロントエンド:
Thymeleaf（サーバーサイドテンプレート）
HTML/CSS/JavaScript（static/ 配下）
anime.js（ローカル UMD ビルド）

バックエンド:
Spring Boot 3.2.2
Spring MVC
Spring Security
Bean Validation（Jakarta Validation）
MyBatis（XML Mapper）
Spring AOP（ログ/性能計測）
springdoc-openapi（Swagger UI）
データベース：H2（インメモリ）

その他:
Java 17
Maven
多言語対応（MessageSource + messages/ValidationMessages）
