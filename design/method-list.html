4 共通処理・ユーティリティ

例外・認証・変換 任意
対象 役割
ApiExceptionHandler 業務例外と Bean Validation 例外を ErrorResponse に整形する
SecurityConfig 公開ページ許可、/staff/** 保護、フォームログイン設定を行う
ReservationSlotService.toResponse(slot) 予約枠モデルを API 応答 DTO に変換する
StaffReservationService.buildComparator(sortBy) 一覧ソート条件に応じた Comparator を生成する
StaffReservationService.readStringMap / readIntegerMap JSON 列を Map に復元する
StaffReservationService.buildVisitTypeCode(name) 来店種別コードを一意に生成する
バリデーション関連 任意
対象 検証内容 メッセージ例
QuestionnaireResultForm.routeCode 必須、32文字以内 routeCode is required
step1Answers / step2Answers 必須、key/value ごとの空欄・長さ制限 step1Answers value must not be blank
graphAxes 必須、各値は 0..100 graphAxes value must be between 0 and 100
ReservationForm.slotId ほか 必須 slotId is required
ReservationForm.staffMemo 500文字以内 staffMemo must be 500 characters or less
予約業務検証 来店種別存在、枠存在、枠競合を確認 Selected reservation slot is no longer available
