ダビスタKSP計算ツール

作品概要 必須

ダビスタ97愛好家向けのWebアプリ。仔馬価格・体重・成長型と種牡馬/繁殖牝馬データからKSP候補を逆算し、SPコメント週チェックで絞り込み＆結果をリスト保存。制作1か月、ブラウザChatGPTへのプロンプトのみで実装。

使用技術 必須

フロントエンド：HTML / CSS / JavaScript

その他：ChatGPT

工夫した点・アピールポイント 必須

ブラウザChatGPTと文言のみのやり取りでレイアウトをイメージ通りの大きさに設置、配置するのに苦労しました。
自身が本ゲームの愛好家であったため、どの部分がユーザーにとって価値あるかを意識して設計しました。
その中でもプルダウンを活用し特定の馬名を逆引きすることが可能になった部分は操作感としてはかなり満足いける作りになったと思います。
