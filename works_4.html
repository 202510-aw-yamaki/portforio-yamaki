制作物概要 必須

本ゲームはキーボード入力タイプのPKゲームアプリです。
Spring Boot + Thymeleafを利用しています。
旧実装としてコンソール版も同梱されています（Main.java）。

使用技術 必須

フロントエンド:
Thymeleaf
Vanilla JavaScript
HTML/CSS

バックエンド：
Java 17
Spring Boot 3.5.10
Spring Web（REST API）
Thymeleaf連携（画面描画）
Spring Validation（依存あり）
Maven（ビルド管理）

その他：
JUnit 5 + Spring Boot Test（contextLoads テスト）
Lombok（依存定義あり）
Spring Boot DevTools（開発時ホットリロード用）
静的アセット（keeper画像）を static 配下で配信

工夫した点・アピールポイント 必須

まずは、VScode上のターミナルのみで完結するJAVAアプリとして制作しました。
その後、springbootのフレームワークを使用し、Webアプリとして再構築しました。
